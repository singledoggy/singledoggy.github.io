fig.add_axes() 方法用于在图形对象中添加一个新的坐标轴。它需要一个四元组参数 [left, bottom, width, height] 来指定新增坐标轴的位置和大小，其中：

  • left：坐标轴左侧边缘相对于图形左侧边缘的偏移量（取值范围为 0 到 1）；
  • bottom：坐标轴底部边缘相对于图形底部边缘的偏移量（取值范围为 0 到 1）；
  • width：坐标轴的宽度（取值范围为 0 到 1）；
  • height：坐标轴的高度（取值范围为 0 到 1）。

在这里，我们使用了一个列表 [0.9, 0.25+0.5*i, 0.02, 0.25] 来指定三个颜色条的位置和大小。其中：

  • 0.9 表示颜色条左侧边缘相对于整个图形左侧边缘的偏移量为 0.9，即颜色条位于整个图形的右侧；
  • 0.25+0.5*i 表示颜色条底部边缘相对于整个图形底部边缘的偏移量，其中 i 是循环变量，用于确定第几个子图；
  • 0.02 表示颜色条的宽度；
  • 0.25 表示颜色条的高度。

Secretary Problem--秘书问题的最佳停止时间

Posted on 2023-03-07

转自：阮一峰

评价：这个问题可以应用到方方面面，确实可以指导我们的生活。

数学家对这个问题，已经有了充分的讨论。经过计算，成功概率最大的方法，叫做"1/e 法则"。e 是自然对数的底数，约等于2.718，那么 1/e 就约等于37%，所以它又称"37%法则"。

"37%法则"的意思是，100个应聘者，先面试前37个，且这37无论多优秀都不取，此后的面试只要遇到一个更优秀的，就立刻录取，停止面试，最后找到最佳候选人的的成功率也为37%。

真硬核赛博朋克还是得练打字。

最高目标大概最高60WPM，但是我觉得40WPM已经能达成效能提升100%了，效益应该是短期最高的。下次提升再来记录感触。
