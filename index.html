——《文心雕龙》

满江红

Posted on 2023-01-27
灯影绰约
电影名: 满江红(2023)
导演: 张艺谋
主演: 沈腾/易烊千玺
上映时间: 2023-01-22(中国大陆)
评分: 7.5

因为看到张艺谋和沈腾，加上有朋友说不错就去看了，超乎想象的烂片。整体我打分只能给3/10，中途无数次想走人。

列举一些缺点：

  • 太长， 159分钟，全程尿点无干货，剧本稀烂。如果想搞反转，起码要有些起转承合，让人有思考推理的地方，然而没有。突兀的反转，突兀的揭晓"大秘密"。
  • 找线索的配乐处没有字幕，虽然能听出来是戏剧，但是听不清唱什么。默认所有人都能听懂听清，我不理解。完全是自娱自乐，但凡找个人看一下立马会觉得这段没有字幕听不清唱什么。
  • 感情线太尴尬，煽情处尬煽情，感动不了人
  • 出戏感太强，与其说是故事不如说是小品，常常有剧情和表演在用力过猛的提醒你这不真实，是演的，表演痕迹太重了
  • 最后读课文尬到抠脚，歌伎、马夫、大头兵，前赴后继的死人，只是为了岳飞最后遗言传出，不想杀秦桧，真的是某些人对底层老百姓的想象，符合某些矫揉做作的文艺分子的想法。

“对于工农兵群众，则缺乏接近，缺乏了解，缺乏研究，缺乏知心朋友，不善于描写他们；倘若描写，也是衣服是劳动人民，面孔却是小资产阶级知识分子。他们在某些方面也爱工农兵，也爱工农兵出身的干部，但有些时候不爱，有些地方不爱，不爱他们的感情，不爱他们的姿态，不爱他们的萌芽状态的文艺（墙报、壁画、民歌、民间故事等）。”

pdf-copy-vim

Posted on 2022-12-07 In Code

为了解决从pdf等格式的中文文献中复制粘贴常会出现的一些问题：

  • 数字、字母等非中文字符变成全角字符
  • 产生多余的空格、换行
  • 中、英文标点（全、半角）混乱
  • 英文上下文中的标点后缺空格

已有一些现有思路，例如利用quicker和打包成GUI界面的软件。由于我使用MacOS和Vim编辑器，为了方便起见，可以利用现有的python脚本在改为vim的插件。

主要需要实现的点是python与vim的交互，可以使用vim.eval('@+')读取寄存器作为参数，输出结果到当前buffer即可。
